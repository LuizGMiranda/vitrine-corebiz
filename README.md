## Iniciar o projeto
Apos fazer o clone do projeto, instale as dependencias:
```bash
npm intall
# or
yarn
```
E para inicar o projeto:
```bash
npm run dev
# or
yarn dev
```
O projeto sera aberto na URL [http://localhost:3000](http://localhost:3000)
